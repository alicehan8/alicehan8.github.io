Assignment 1: Data Collection and Analysis

Using the data collected by the class, I performed an analysis to determine the optimal location to wait for the elevator in order to minimize the distance traveled. In order to do so, I used the Python's pandas package for data manipulation and analysis, as well as matplotlib for data visualization. By calculating elevator arrival probabilities, I was able to identify the most efficient waiting point.

Assignment 3: Singular Value Decomposition

In this assignment, I created a Singular Value Decomposition (SVD) from scratch. Using that SVD function, I preprocessed the MNIST dataset, which then trained a logistic regression classifier. I then compared the model performance based on accuracy and training time to determine the optimal number of SVD components.

Assignment 4: Latent Semantic Analysis (LSA) Search Engine

In this assignment, I implemented a web application that acted as a basic search engine using LSA. It takes a query from the user, performs LSA on a pool of documents, and returns the top 5 most similar docuemnts.
